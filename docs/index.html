Agent-first artifact host.

Any terminal agent generates HTML; hart gives it a live, sandboxed, versioned URL. One static machin binary is both the CLI and the daemon. BYOK, open-core, self-hostable.

Sandboxed by default

Untrusted agent HTML is served under a strict CSP (default-src 'none', no external or network access) with frame/referrer hardening.

Content-addressed versions

Every publish is immutable and hash-addressed; latest moves, old pins stay. Roll back to any v<n> forever.

Agent-first I/O

JSON in, JSON out; semantic exit codes; no UI tax. Built for Cursor, aider, Codex, Gemini CLI, or your own.

The whole contract is one POST. The page is the request body; id/title/format are query params.

publish an artifact
hart publish report.html --title 'Q3 numbers'
# -> id a1b2, version 1, https://hart.intrane.fr/a/a1b2/latest

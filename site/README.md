# hart site — built with machin-web-ui

hart's landing page, composed from [machin-web-ui](https://github.com/javimosch/machin-web-ui)
components and themed with hart's teal brand (`theme.json`). An isomorphic
machin app: SSR first paint + a wasm client that hydrates the static markup.
No Node.

```sh
MACHIN_WEB_UI=/path/to/machin-web-ui ./build.sh   # needs machin + zig too
./app --port 48150                                # http://localhost:48150
machin-web-ui verify --url http://localhost:48150/  # prove it renders (pure-MFL CDP)
```

Rebranding is `theme.json`: ten semantic tokens mapped to hart's palette
(`accent: #3ad0c4`, dark `canvas`/`surface`, light `ink`). The whole stylesheet
is generated by machin-web-ui's Tailwind engine and oracle-verified.
